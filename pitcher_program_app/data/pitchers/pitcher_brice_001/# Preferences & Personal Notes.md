# Preferences & Personal Notes

## Athlete Profile
- College pitcher, University of Chicago varsity baseball
- Junior (2025-2026 academic year)
- In-season starter on 7-day rotation

## What Has Worked
- Dry needling for forearm/UCL area tightness — highly effective
- Pronator strengthening as injury prevention baseline
- Rocksauce pre-throw to activate forearm
- Soft tissue work before all arm care sessions
- Nerve glides for medial elbow symptom management

## What to Avoid / Red Flags
- Skipping pronator work when arm feels fine (leads to regression)
- Aggressive valgus stress exercises when medial elbow is symptomatic
- High-intent throwing when forearm tightness is present — always back off
- Rushing return to full throwing after any tightness episode

## Protocol Preferences
- Prefers structured day-by-day format after each outing
- Wants protocols to adjust based on: pitch count, arm feel, HRV/recovery data, days until next outing
- Values knowing the *why* behind each exercise recommendation
- Likes having a "modified" protocol option for days when arm feels off

## Communication Style
- Be direct and specific — no generic advice
- Flag when symptoms suggest backing off vs. pushing through
- If forearm tightness or UCL-area sensation is reported, default to conservative protocol and note it explicitly
