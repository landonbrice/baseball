## Arm Care Heavy Program — Pronator Focus

**Pre-requisite:** Perform soft tissue work first as needed.

**Format:** Supersets — matching letters performed back-to-back, then repeat.

| Superset | Exercise | Reps |
|----------|----------|------|
| A1 | Wall Sit Serratus Scoop | 3x10 |
| A2 | Front Plank Hand Taps | 3x6-8 each |
| B1 | Supine Bilateral External Rotation | 2x10 |
| B2 | Eccentric Internal Rotation | 3x10 |
| C1 | Supine T | 2x10 |
| C2 | Pec Pallof | 2x12 |
| D1 | Prone Snow Angel | 3x8 |
| D2 | Thoracic Rotation Eccentric Pec (or Eccentric Pec Lowering Heavy Band) | 3x8 |
| E1 | Full Pronation | 3x12 |
| E2 | Axial Rotation Pronation-Supination | 3x15 |
| F1 | Shaker Ball Series (circles, alphabet, throwing motion) | 2x30sec (circles or alphabet); 3 throwing motions |

### Exercise Categories Represented:
- **Scapular stability**: Wall sit serratus scoop, prone snow angel, supine T
- **Core/anti-rotation**: Front plank hand taps, pec pallof
- **Shoulder rotation**: Supine BL ER, eccentric IR
- **T-spine mobility**: Thoracic rotation eccentric pec
- **FPM/Pronator**: Full pronation, axial rotation pronation-supination
- **Neuromuscular**: Shaker ball series
