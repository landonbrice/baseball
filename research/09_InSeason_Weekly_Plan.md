# 09 — In-Season Weekly Plan (Pitcher)

_Last updated: 2026-03-14_

> Built from the research dossier (maintenance of strength/power in-season, reduced volume, strategic timing around outings).

## How to use this file
- Pick the track that matches your role this week:
  - **Track A:** Starter (7-day)
  - **Track B:** Starter (6-day)
  - **Track C:** Reliever (auto-regulated)
- Keep **1–3 reps in reserve (RIR)** on almost everything in-season.
- If arm/overall fatigue rises: cut volume by **20–40% first**, not intensity.

---

## Track A — Starter (7-day)\
_(Example: game day = Saturday)_

### D0 — Game day
- Pitch
- Post-game flush (10–15 min): bike/walk + light mobility + forearm tissue + breathing reset

### D+1 — Lift A (highest stress, 35–60 min)
1. Trap bar deadlift **or** front squat — 4 x 3–5 @ RPE 7–8
2. Rear-foot elevated split squat — 3 x 5–8/side
3. Chest-supported row or 1-arm DB row — 3 x 6–10
4. Landmine press or DB bench — 2–3 x 4–8
5. Cuff/scap circuit (2 rounds):
   - Band ER x 12–15
   - Face pull x 12–15
   - Serratus wall slide x 8–12

### D+2 — Recovery + optional low power
- Mobility 15–20 min (t-spine, hips, lats)
- Cuff/scap + forearm care 15 min
- Optional med-ball rotational throws — 3 x 3/side (submax)

### D+3 — Lift B (moderate + power, 30–50 min)
1. Jump variation (box jump or broad jump) — 4 x 2–3
2. RDL **or** front squat (lighter than Lift A) — 3 x 3–6 @ RPE 6–7
3. Pull-up/chin-up (or lat pulldown) — 3 x 4–8
4. Med-ball scoop/shotput rotational — 3–5 x 3/side
5. Forearm block — 2 sets each:
   - Pronation/supination
   - Wrist extension
   - Grip crush/carry

### D+4 — Low day
- Walk + mobility + tissue work only
- No fatigue-chasing lift

### D+5 — Primer (20–30 min, neural only)
- Jumps — 2 sets
- Med-ball rotational — 2–3 sets
- Light lower pattern — 1–2 sets of 3
- Light pull pattern — 1–2 sets of 4–6

### D+6 — Pre-start readiness
- Warm-up quality, activation, throwing prep
- No meaningful gym fatigue

## Track B — Starter (6-day)

### Template
- **D0:** Game
- **D+1:** Lift A (brief heavy)
- **D+2:** Recovery + arm care
- **D+3:** Lift B (power + maintenance)
- **D+4:** Mobility + low CNS
- **D+5:** Primer / bullpen integration

### Lift A (brief heavy, 30–45 min)
1. Main lower — 3 x 3–5 @ RPE 7–8
2. Unilateral lower — 2 x 5–8/side
3. Pull — 3 x 6–8
4. Press — 2 x 4–8
5. Cuff/scap — 1–2 rounds

### Lift B (power + maintenance, 25–40 min)
1. Jump — 3–4 x 2
2. Hinge/squat variant — 2–3 x 3–6 @ RPE 6–7
3. Pull-up/row — 2–3 x 5–8
4. Med-ball rotational — 3 x 3/side
5. Forearm/grip — 1–2 sets

---

## Track C — Reliever (auto-regulated)

### Rules
- **High-stress outing <24h:** recovery only
- **No outing for 48–72h:** one concise full-body lift
- **No outing >72h:** two concise sessions

### Concise full-body lift (25–40 min)
1. Main lower — 3 x 3–5
2. Pull — 3 x 5–8
3. Press/landmine — 2–3 x 4–8
4. Jump or med-ball — 2–4 sets
5. Cuff/scap — 1–2 rounds

Weekly target: **2–3 brief lifts** depending on usage pattern.

---

## Exercise substitution bank
- Main lower: trap bar DL, front squat, safety bar squat
- Hinge: RDL, barbell hip hinge, DB RDL
- Pull: pull-up, chest-supported row, cable row
- Press: landmine press, DB incline, push-up variants
- Power: jumps, bounds, med-ball rotational scoop/shotput

---

## Readiness / shutdown rules
- If mechanics/pain change, stop and regress workload same day.
- If soreness + dead-arm trend >48h, remove one lift slot that week.
- If game stress spikes, preserve intensity and trim total sets first.
