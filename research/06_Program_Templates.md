# 06 — Program Templates (Practical)

> These are evidence-informed templates, not medical advice. Adjust for age, injury history, calendar, and pitching workload.

## Template 1: In-season starter (7-day) — “Maintain + Perform”

### Day-by-day
- **D0 (Start):** game; post-game recovery circuit 10–15 min
- **D+1:** Lift Heavy-Moderate (35–60 min)
- **D+2:** Recovery + mobility + cuff/scap + optional low-volume med-ball
- **D+3:** Lift Moderate + power touch
- **D+4:** Low day (walk, mobility, tissue)
- **D+5:** Primer (short neural session)
- **D+6:** Pre-start readiness only

### Lift Heavy-Moderate
1. Main lower (trap bar DL/front squat): 4 x 3–5 @ RPE 7–8
2. Secondary lower unilateral: 3 x 5–8
3. Main pull: 3 x 6–8
4. Main press (if tolerated): 2–3 x 4–8
5. Cuff/scap finisher: 2 rounds

### Lift Moderate + Power
1. Jump or bound: 4 x 2–3
2. Hinge/squat variant: 3 x 3–6 @ RPE 6–7
3. Pull-up/row superset: 3 x 5–10
4. Med-ball rotational throw: 3–5 x 3/side
5. Forearm/grip: 2–3 sets

### Primer
- 2 jump sets
- 2–3 med-ball sets
- 1–2 light lift sets each for lower + pull
- done in 20–30 min, no fatigue chasing

## Template 2: In-season reliever — “Auto-regulated micro-dose”

### Rules
- If high-stress appearance in last 24h: recovery-only
- If no appearance for 48–72h: perform one concise total-body session
- If no appearance >72h: two sessions in that window

### Concise total-body session (25–40 min)
1. Main lower: 3 x 3–5
2. Pull: 3 x 5–8
3. Press/landmine: 2–3 x 4–8
4. Jump or med-ball: 2–4 sets
5. Cuff/scap: 1–2 rounds

## Template 3: Off-season (4-day) — “Strength to velocity bridge”

### Monday (Lower Strength)
- Squat pattern 5 x 3–5
- RDL 3 x 5–8
- Split squat 3 x 6
- Jumps 4 x 2
- Trunk anti-rotation 3 sets

### Tuesday (Upper Strength + Rotational Power)
- Bench/DB press 4 x 4–6
- Pull-up 4 x 4–8
- Row 3 x 6–10
- Med-ball rotational 5 x 3/side
- Cuff/scap 3 blocks

### Thursday (Lower Power)
- Olympic derivative or jump squat 5 x 2–3
- Hinge 3 x 4–6
- Unilateral lower 3 x 6–8
- Sprint or acceleration work 4–8 reps

### Friday (Upper Power + Arm Capacity)
- Push press/landmine 4 x 3–5
- Pull variation 4 x 5–8
- Med-ball scoop/shotput 4–6 sets
- Forearm pronation/supination + wrist extension/flexion 2–3 sets each
- Cuff/scap finisher

---

## Template 4: Off-season (3-day total body) — limited schedule athlete

### Day 1 (Strength)
- Main lower 4 x 3–5
- Main upper pull 4 x 5–8
- Accessory push/pull 2–3 sets each
- Med-ball 3–4 sets

### Day 2 (Power)
- Jump + sprint micro-combo
- Main hinge 3 x 3–5
- Upper press/pull 3 x 4–8
- Cuff/scap block

### Day 3 (Mixed)
- Secondary squat 3 x 4–6
- Unilateral lower 3 x 6
- Row/pull-up 3 x 5–8
- Rotational med-ball 4 x 3/side
- Forearm/grip block

---

## Progression & autoregulation (all templates)
- Week-to-week: add 2.5–10 lb only when intent and recovery are good.
- If soreness/fatigue rises or game stress spikes: reduce 20–40% volume before touching intensity.
- Keep 1–3 reps in reserve most sets in-season.
- Pain that alters mechanics = immediate modification + medical/sports med follow-up.
