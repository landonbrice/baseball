# 11 — Gaps to Operationalize (Adaptive Lifting System for Pitchers)

_Last updated: 2026-03-14_

> Purpose: turn the evidence gaps into a **working decision system** so your plan adapts to real readiness, not fixed templates.

## 1) Daily Readiness Dashboard (2–3 min)

Score each item 0–2. Max = 12.

### Inputs
1. **Sleep quality/quantity**
- 2 = normal/good
- 1 = slightly down
- 0 = poor/short

2. **Arm soreness (medial elbow/shoulder/forearm)**
- 2 = minimal
- 1 = noticeable but stable
- 0 = sharp/increasing/changes mechanics

3. **Whole-body fatigue**
- 2 = fresh
- 1 = moderate
- 0 = heavy fatigue

4. **Warm-up velocity/intent feel**
- 2 = normal/pop
- 1 = slightly flat
- 0 = clearly down

5. **Mood/motivation/CNS feel**
- 2 = normal high
- 1 = neutral
- 0 = unusually flat/irritable

6. **Objective marker (pick one):**
- Countermovement jump / med-ball speed / bar-speed snapshot
- 2 = normal
- 1 = 5–10% down
- 0 = >10% down

### Readiness bands
- **10–12 (Green):** run planned session
- **7–9 (Yellow):** reduce volume 20–30%, keep quality
- **4–6 (Orange):** technique/power only, no grinding strength
- **0–3 (Red):** recovery day + arm-care only

---

## 2) Auto-Adjustment Rules (Session Level)

### Strength lift days
- If **Yellow**: remove last set from each main movement.
- If **Orange**: main lift 2–3 sets @ RPE 6–7 only; skip accessories.
- If **Red**: no loaded main lift.

### Power/primer days
- Keep only high-quality jump/med-ball reps.
- Stop when output drops >10% from best rep.

### Throwing + lifting same day
- Throw quality first.
- Lift volume capped to 30–40 min unless Green.

---

## 3) Weekly Load Guardrails

### Hard constraints
- Never increase **both** throwing intensity and lifting volume in same 7-day block.
- Max 2 high-CNS gym sessions in-season (starter model).
- Keep at least 1 strength exposure/week unless Red trend persists.

### If symptoms trend up for 2+ days
- Cut gym set volume by 30–40% for 5–7 days.
- Keep movement speed and quality.
- Emphasize cuff/scap + forearm and tissue recovery.

### If readiness trend is high for 2+ weeks
- Add one set to primary lower and primary pull lifts only.
- Keep accessory volume stable.

---

## 4) Role-Specific Auto Rules

### Starter
- Highest lift stress on **D+1** only if Green/Yellow.
- D+3 stays moderate; never turn D+3 into max-strength day.

### Reliever
- Lift decisions triggered by appearance pattern:
  - high-stress appearance <24h = recovery only
  - no appearance 48–72h = concise full-body

### Two-way athlete
- Use reliever rules + stricter CNS cap:
  - 1 heavy lower session/week max in-season.

---

## 5) Return-to-Throw / Return-to-Lift Bridge

When coming off arm flare-up:
1. 72h symptom stability at low intensity
2. Reintroduce throwing progression first
3. Reintroduce lifting as:
   - Week 1: 60–70% volume
   - Week 2: 75–85% volume
   - Week 3: normal if no symptom trend

If pain changes mechanics at any step, regress immediately.

---

## 6) Minimum Data to Track (Weekly)

Log these each week in one line:
- Avg readiness score
- # high-intent throw days
- # lift sessions completed
- Best velocity (or intent proxy)
- Arm symptom trend (better/same/worse)
- Any missed sessions due to fatigue/pain

This creates your personal dataset so the plan gets smarter over time.

---

## 7) 4-Week Microcycle Decision Logic

### Week 1: Establish baseline
- Run planned split with dashboard only (no additions).

### Week 2: Stabilize
- Keep same structure, adjust daily by readiness bands.

### Week 3: Nudge
- If readiness stable + symptoms neutral, add 1 set to main lower OR pull.

### Week 4: Deload
- Reduce total volume 30–40%; keep speed/intent high.

Repeat cycle.

---

## 8) What this operational layer solves
- Converts vague “listen to your body” into **clear thresholds**.
- Handles research uncertainty on exact dose via adaptive control.
- Protects arm health while preserving performance qualities.
