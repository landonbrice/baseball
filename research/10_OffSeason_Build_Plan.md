# 10 — Off-Season Build Plan (Pitcher)

_Last updated: 2026-03-14_

> Goal: build force + power that transfers to mound velocity while protecting shoulder/elbow capacity.

## Structure options
- **Option 1 (recommended):** 4-day split (higher capacity)
- **Option 2:** 3-day total-body (time-limited weeks)

## Global rules
- Main lifts: stop with **1–2 RIR** (don’t grind ugly reps)
- Progress load only when bar speed/intent stays high
- Keep rotational power and throwing-specific intent in the week
- Include cuff/scap + forearm every week (not optional)

---

## Option 1 — 4-day split (12-week template)

### Weekly split
- **Mon:** Lower Strength + Jumps
- **Tue:** Upper Strength + Rotational Power
- **Thu:** Lower Power + Speed
- **Fri:** Upper Power + Arm Capacity

### Monday — Lower Strength
1. Squat pattern (front/safety/trap) — 5 x 3–5
2. RDL — 3 x 5–8
3. Split squat — 3 x 6/side
4. Jump (box/broad) — 4 x 2
5. Anti-rotation trunk — 3 sets

### Tuesday — Upper Strength + Rotation
1. DB bench or incline — 4 x 4–6
2. Pull-up/chin-up — 4 x 4–8
3. Row variation — 3 x 6–10
4. Med-ball rotational scoop/shotput — 5 x 3/side
5. Cuff/scap block (3 moves x 2 rounds)

### Thursday — Lower Power
1. Olympic derivative or jump squat — 5 x 2–3
2. Hinge variation — 3 x 4–6
3. Unilateral lower — 3 x 6–8/side
4. Sprint/acceleration — 4–8 reps (10–20m)

### Friday — Upper Power + Arm Capacity
1. Push press or landmine press — 4 x 3–5
2. Pull variation — 4 x 5–8
3. Med-ball overhead/scoop blend — 4–6 sets total
4. Forearm block — 2–3 sets each:
   - Pronation/supination
   - Wrist extension/flexion
   - Grip/carry
5. Cuff/scap finisher — 2 rounds

---

## Option 2 — 3-day total-body (time-limited)

### Day 1 (Strength)
1. Main lower — 4 x 3–5
2. Main upper pull — 4 x 5–8
3. Accessory push/pull — 2–3 sets each
4. Med-ball rotational — 3–4 sets

### Day 2 (Power)
1. Jump + short sprint combo — 4–6 rounds
2. Main hinge — 3 x 3–5
3. Upper press/pull — 3 x 4–8
4. Cuff/scap block — 2 rounds

### Day 3 (Mixed)
1. Secondary squat — 3 x 4–6
2. Unilateral lower — 3 x 6/side
3. Row/pull-up — 3 x 5–8
4. Rotational med-ball — 4 x 3/side
5. Forearm/grip block — 2 sets each

---

## 12-week progression (simple)

### Block 1 (Weeks 1–4): Capacity + technique
- Main lifts in 5–8 rep emphasis
- Moderate loads, clean movement quality
- Build tolerance for weekly volume

### Block 2 (Weeks 5–8): Max strength emphasis
- Main lifts 3–5 reps
- Slightly lower accessory volume
- Keep jumps/med-ball in every week

### Block 3 (Weeks 9–12): Power transfer
- Main lifts 2–4 reps with intent
- More ballistic/med-ball quality
- Total gym fatigue lower, speed quality higher

### Deloads
- Every 4th week: reduce volume by ~35–45%, keep movement speed crisp

---

## Throwing integration notes
- Pair higher CNS throwing days with higher CNS lift days when possible
- Keep low days truly low (mobility/recovery/skill)
- If elbow/shoulder symptoms increase, reduce total high-intent exposures for 5–7 days

---

## Minimum arm-care checklist each week
- Cuff/scap: 2–3 sessions
- Forearm pronation/supination: 2–3 sessions
- Thoracic/lat mobility: 3+ short sessions
- Sleep/recovery monitoring: daily trend check
